**Water Tracker** is an Android application for tracking daily water intake.  
It helps users set hydration goals, log consumed water, and receive reminders to stay hydrated throughout the day.

## Features
- Set daily water intake goals
- Log consumed water easily
- Track daily hydration progress
- Smart reminders to drink water
- Clean and minimal UI

## Tech Stack
- **Language:** Kotlin  
- **UI:** Jetpack Compose  
- **Architecture:** MVI  
- **DI:** Dagger Hilt  
- **Database:** Room  
- **Background Tasks:** WorkManager  
- **Navigation:** Voyager  
- **Notifications:** Android Notifications  

## App Behavior
- Sends scheduled reminders to drink water during the day
- Saves daily water consumption locally
- Automatically resets progress every day
